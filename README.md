# calculatorwithpython
This program creates a simple graphical calculator (like the one on your phone, but made in Python). It has a screen at the top where numbers and results show up, and a grid of buttons below for digits (0-9), operations (+, −, ×, ÷), a Clear button, and an Equals button.
Everything is built using Tkinter, which is Python’s built in tool for making windows, buttons, and text fields. The use just has to use it the same way he uses his calculator in his phone. He has to choose the numbers, the  arithmetic operator and equal for outputting the result.
